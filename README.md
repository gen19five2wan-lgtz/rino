# 居酒屋 里野 (Izakaya Rino) - 公式ウェブサイト

東京都武蔵野市西久保に位置する居酒屋「里野（りの / Rino）」の公式ホームページです。

## 概要

BS-TBS『吉田類の酒場放浪記』で紹介された、人情薫る大人の隠れ家。
新鮮な本マグロと懐かしの昭和洋食が自慢の地域密着型居酒屋です。

## 特徴

- 📍 三鷹駅からバスで便利（三谷通り下車 徒歩2分）
- 🐟 市場直送の新鮮な魚（本マグロが看板メニュー）
- 🍝 懐かしの昭和洋食（ナポリタン、里野ピザ）
- 🏮 温かみのある昭和レトロな雰囲気
- 💴 現金のみ対応
- 🚭 屋内禁煙・屋外喫煙可

## ローカル開発

```bash
# HTTPサーバーを起動
npx http-server -p 8080

# または
python -m http.server 8080
```

ブラウザで `http://localhost:8080` を開いてください。
